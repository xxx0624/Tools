# Baidu Search

## specially for baike item

You can use this function 'get_baike_search_result(keywords)'

given one word, return the part of the baike's item(Or you can change the paser to get all the html content)

eg. given "方言(佟大为饰演角色)" <br>
it will return the website(http://baike.baidu.com/subview/16035/10799068.htm#viewPageContent) body

eg. given "加农贝克" <br>
it will return the website(http://baike.baidu.com/view/2675459.htm) body

PS: if the search word has many items, you'd better add the brackets and its content(please be sure it's right format)
